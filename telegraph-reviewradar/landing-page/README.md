# ReviewRadar Landing Page

Single-page React + TypeScript landing page for **ReviewRadar**.

The page includes:
- Glassmorphism UI with floating navigation
- Mouse-reactive animated background
- Multi-section layout with grouped full-screen scroll flow
- Supported platform cards (Amazon, eBay, Google Play, App Store)
- Feature, benefits, FAQ, and CTA sections

## Tech Stack

- React 19
- TypeScript
- Vite
- Lucide React icons

## Run Locally

```bash
npm install
npm run dev
```

Open the dev server URL shown in terminal (usually `http://localhost:5173`).

## Build

```bash
npm run build
npm run preview
```
